# Personal Website

This is the source code of my personal website. I built it using only HTML, CSS and JS. 

The first website I ever made and I still use it as my online resume.

The other projects on my profile use more advanced technologies. But this is where I started. 

It serves as a reminder to myself and anyone who happens to chance upon this repo that 

- The journey of a thousand miles begins with a single step. (For a dev a single page website)

- That there is no running before you can walk. (No React or Vue before you can write Vanilla JS)

- Getting the fundamentals right is the key to success. (Think like Warren Buffett)

- The slowest guy in the race has still beat everyone on the couch.

Enough motivation for now. 

**Go do the thing that scares you.**
